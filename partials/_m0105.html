Vagrant and Virtualbox
Requirements for installation:
• BIOS supports hardware virtualization
• At least 5GB of drive space
• 512MB RAM allocated to VMs
• Hyper-V disabled in Windows
  In a PowerShell administrator prompt, type:
    Disable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -All
After installation, verify using these commands:
vagrant --version
VBoxManage --version
Install the CentOS 7.2 Box
In the SSH client, run:
vagrant box add bento/centos-7.2 --provider=virtualbox
Launch the CentOS 7.2 Box
In the SSH client, run:
To create the Vagrantfile: vagrant init bento/centos-7.2
To start the VM: vagrant up
Log In to the CentOS 7.2 Box
In the SSH client, run:
vagrant ssh - if there are more than one VM, the VM name is needed, e.g. 'web1'
Install the ChefDK
In the SSH client, run:
curl -L https://omnitruck.chef.io/install.sh | sudo bash
Common Vagrant Commands
  • vagrant ssh-config
- displays connection details
  • vagrant init
- creates a Vagrant file
  • vagrant status
- lists VMs and their status
  • vagrant suspend
- saves VM state and shuts down
  • vagrant destroy --force
- destroys all running VMs
