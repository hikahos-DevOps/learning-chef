Knife
A command-line tool that provides an interface between a local chef-repo and the Chef Server
In the SSH client, run:
knife client list
It should return the name of your organization with the '-validator' suffix
Inside the 'chef-repo' directory, there is a hidden '.chef' directory that contains pem file or private key
Common Knife commands:
knife cookbook list
 knife cookbook upload [COOKBOOK]
