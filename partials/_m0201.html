Chef Resource
A way to manage a system component
A statement of configuration policy
It describes the desired state of an element of your infrastructure and the steps needed to bring that item to the desired state.
Example: Package
package 'httpd' do
  action :install
end
Example: Service
service 'ntp' do
  action [ :enable, :start ]
end
Example: File
file '/etc/motd' do
  content 'I am the message of the day!'
end
Example: File
file '/etc/php.ini.default' do
  action :delete
end
Resource Definition
file '/hello.txt' do
  content 'Hello, world!'
  action :create
end
 The TYPE (file) named NAME ('/hello.txt') should be ACTION'd (:create) with PROPERTIES ('Hello, world!').
