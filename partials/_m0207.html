Ohai
Ohai is a tool that already captures all the data about system resources.
Example Ohai commands:
ohai ipaddress
 ohai hostname
 ohai memory
All About the System
• Ohai queries the operating system with a number of commands.
• The data is presented in JSON format.
Ohai + chef-client
chef-client and chef-apply automatically executes ohai and stores the data about the node in an object you can use within the recipes named node.
The Node Object
The node object is a representation of our system. It stores all the attributes found about the system.
Node Details in the Webserver
• The file resource named '/var/www/html/index.html' can be updated with the content that includes the node details:
  ipaddress
  hostname
Update the Apache server.rb Recipe
...

file '/var/www/html/index.html' do
  content "<h1>Hello, world!</h1>
  <h2>ipaddress: #{node['ipaddress']}</h2>
  <h2>hostname: #{node['hostname']}</h2>"
end
