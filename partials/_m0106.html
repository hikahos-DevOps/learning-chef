Chef-client
chef-client is an agent that runs locally on every node that is under management by Chef.
It will perform all of the steps that are required to bring the node into the expected state.
--local-mode (or -z)
The default mode for chef-client attempts to contact a Chef Server for the recipes to run.
Local mode overrides that behavior.
Run the Recipe
sudo chef-client --local-mode hello.rb
