ChefDK
The Chef Development Kit contains tools like chef generate, chef-client, knife, and kitchen.
You can find the ChefDK to download at: downloads.chef.io
ChefDK is a tool chain built on top of the Ruby language.
The ChefDK installer does not install any particular graphical-user-interface -- installs CLI instead
Lab Commands
$ chef --version
$ chef-client --version
$ knife --version
$ ohai --version
$ berks --version
$ kitchen --version
$ foodcritic --version
$ cookstyle --version
