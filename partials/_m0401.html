Knife Node Commands
In the SSH client, run this command to show details of a node:
kitchen node show NAME
add '-a' to specify a node attribute:
kitchen node show NAME -a ipaddress
 kitchen node show NAME -a hostname
 kitchen node show NAME -a memory.total
 kitchen node show NAME -a cloud.public_ipv4
