Bootstrapping a Node
Often, the node you are bootstrapping may not have the chef-client installed.
It may also not have details of where the Chef Server is located or the credentials to securely talk to that server.
To add those credentials, you can bootstrap that node to install all those components.
In the SSH client, run this command to bootstrap a node:
knife bootstrap FQDN -x USER -P PWD --sudo -N node1 -r "recipe[cookbook]"
In the SSH client, run this command to get details of a bootstrapped node:
knife node show node1
