Creating a Webserver Cookbook
You need three Chef resources
  • Package
  • File
  • Service
Apache Server
  • Default webpage, index.html
  • Creates a directory /var/www/html/
  • Defaults to port 80
  • You need Chef resources to create the directory and start the service, httpd
Create a New Recipe
chef generate recipe cookbooks/apache server

server.rb
package 'httpd'

file '/var/www/html/index.html' do
  content '<h1>Hello, world!</h1>'
end

service 'httpd' do
  action [:start, :enable]
end
Converge the Server Recipe in Local Mode
sudo chef-client --local-mode cookbooks/apache/recipes/server.rb
